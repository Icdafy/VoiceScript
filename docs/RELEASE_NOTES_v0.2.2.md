# VoiceScript v0.2.2 发布说明

## 版本定位

- v0.2.2 是基于真实运行截图的界面修复版本，针对 v0.2.1 暴露的图标与排版问题逐项修正。
- 转录、导出、模型逻辑保持不变。

## 修复内容

- **图标渲染**：修复全局图标被裁切到角落、无法完整显示的问题。根因是在绘制前就设置了 `devicePixelRatio`，导致 SVG 渲染视口被压缩；改为先在普通 `QImage` 上绘制再标记 DPR。所有导航、功能卡、设置、表格操作图标现在均完整居中。
- **设置图标**：「设置」由原先类似太阳的简化图形改为清晰的齿轮，避免与深色模式的太阳图标混淆。
- **移除登录 / 注册**：删除侧边栏底部的「登录 / 注册」入口。
- **移除账户入口**：删除右上角「账户」按钮。
- **模块重叠**：修复上传区「选择文件」按钮与下方三张功能卡重叠的问题（拖拽区改为固定高度并以上下弹性居中内容）。
- **整体排版**：主内容区改为可滚动布局，窗口高度不足时自动滚动，在任意窗口尺寸下模块均不再重叠、排列有序。

## 兼容性

- 黑 / 白主题、双模型档位、TXT / Markdown / SRT / JSON 导出与之前一致。
- 用户配置（主题、保存位置）继续沿用。

## 不包含

- 不做总结、会议纪要、语义整理、说话人分离、实时麦克风转写。

## 发布文件

- `VoiceScript-v0.2.2-windows-x64.zip`

模型权重不包含在发布包中。
