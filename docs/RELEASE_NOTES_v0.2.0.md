# VoiceScript v0.2.0 发布说明

## 重要变化

- v0.2.0 为从零重写版本，不沿用 v0.1.4 任何源码。
- 桌面端改为围绕参考图重建：左侧导航、中央上传、右侧设置、最近文件表。
- 接入 Qwen3-ASR 标准/精准双模型档位。
- 使用 Qwen3-ForcedAligner 输出时间分段。
- 支持 TXT、Markdown、SRT、JSON 导出。
- 黑/白主题切换会持久化。

## 不包含

- 不做总结。
- 不做会议纪要。
- 不做语义整理。
- 不做说话人分离。
- 不做实时麦克风转写。

## 发布文件

- `VoiceScript-v0.2.0-windows-x64.zip`

模型权重不包含在发布包中。
