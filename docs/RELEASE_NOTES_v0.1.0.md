# VoiceScript v0.1.0 Release Notes

Initial local-first Windows desktop release.

## Added

- PySide6 desktop app with upload, drag-drop, model selection, black/white themes, progress, cancellation, transcript table, export, and Obsidian save.
- Whisper large-v3 backend.
- Qwen3-ASR-1.7B backend with Qwen3 ForcedAligner timestamps.
- Unified transcript schema and exporters for Markdown, TXT, SRT, and JSON.
- CLI transcription and environment check commands.
- Windows PyInstaller build script.

## Not Included

- Meeting minutes.
- Summaries.
- Semantic rewriting.
- Speaker diarization.
- Real-time microphone transcription.
