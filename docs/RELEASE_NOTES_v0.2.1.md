# VoiceScript v0.2.1 发布说明

## 版本定位

- v0.2.1 是 v0.2.0 之上的界面与交互升级版本，转录、导出、模型逻辑保持不变。
- 目标：全面扁平化设计 + 接近苹果 / 谷歌的丝滑交互体验。

## 界面重做

- 统一的线性矢量图标体系（导航、品牌、功能卡、设置、表格操作），替换旧版文字符号与系统图标。
- 卡片改用克制描边 + 柔和投影，整体更扁平、更通透。
- 右上角新增账户入口，设置项补充信息提示图标与悬停说明。
- 功能卡、下拉菜单、滚动条、提示气泡统一视觉风格。

## 交互升级

- 开关（说话人识别 / 标点恢复）采用缓动动画滑块，轨道颜色平滑过渡。
- 进度条数值平滑过渡，不再跳变。
- 转录结果卡片淡入出现，主窗口启动淡入。
- 上传区在拖拽文件时高亮边框与背景，给出明确反馈。
- 最近文件表加入文件图标、状态徽标、行悬停高亮与行内操作按钮（查看文件 / 打开文件夹 / 更多）。

## 兼容性

- 黑 / 白主题、双模型档位、TXT / Markdown / SRT / JSON 导出与 v0.2.0 一致。
- 用户配置（主题、保存位置）继续沿用，无需迁移。

## 不包含

- 不做总结、会议纪要、语义整理、说话人分离、实时麦克风转写。

## 发布文件

- `VoiceScript-v0.2.1-windows-x64.zip`

模型权重不包含在发布包中。
