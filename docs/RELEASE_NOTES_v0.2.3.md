# VoiceScript v0.2.3 发布说明

## 核心转录升级（最重要）

- **长音频分段转录**：上传的音频按 **30 分钟** 自动切片，逐段送入 Qwen3-ASR，再按时间偏移合并，稳定转录数小时录音，避免长音频丢字。
- **默认中文输出**：未指定语言或选择「自动识别」时默认以中文（Chinese）输出，符合中文录音为主的使用场景。
- **带时间戳文本**：每条片段输出 `[HH:MM:SS - HH:MM:SS] 文本`，TXT / Markdown / SRT / JSON 均带时间戳。
- 已对照官方仓库 <https://github.com/QwenLM/Qwen3-ASR> 复核核心调用：
  `Qwen3ASRModel.from_pretrained(...forced_aligner=...)` 与
  `model.transcribe(audio, language, return_time_stamps=True)` 用法正确，
  时间戳来自 Qwen3-ForcedAligner，逐段对齐后合并。

## 进度显示细化

- 实时显示阶段信息：「正在转录第 N/M 段（00:30:00 – 01:00:00）· 提取音频片段 / Qwen3-ASR 识别中」，
  以及每段完成后的「已转录 mm/总时长 · 累计片段数」。
- 进度条显示百分比。

## 界面调整

- 导航精简为 **首页 / 文件列表 / 历史记录**，移除「设置」与「帮助与反馈」。
- 新增可用的 **文件列表页**（列出保存目录中的转录结果文件，可打开文件 / 目录、刷新）。
- 新增可用的 **历史记录页**（全部转录任务，含状态徽标、行内操作、清空历史）。
- 修复「说话人识别」开关无法开启的问题：开关现可正常切换并作为偏好保存
  （注：当前 Qwen3-ASR 尚未内置说话人分离，开启暂不改变转录结果，待后续接入）。
- 「转录模型」说明更详细：解释标准模型（0.6B，轻量快速，推荐）与精准模型（1.7B，更准但更重）的差异与适用场景。
- **深色模式改为完全深色**：去除卡片浅色描边，仅保留输入框 / 下拉框的细边，整体更黑、更统一。

## 兼容性

- 双模型档位、TXT / Markdown / SRT / JSON 导出与之前一致。
- 用户配置（主题、保存位置）继续沿用。
- 运行需本机安装 FFmpeg（用于音频分段与解码）。

## 不包含

- 不做总结、会议纪要、语义整理、说话人分离、实时麦克风转写。

## 发布文件

- `VoiceScript-v0.2.3-windows-x64.zip`

模型权重不包含在发布包中，首次使用由运行时下载到本机缓存。
