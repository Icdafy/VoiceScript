# 模型说明

## 标准模型

- ASR：`Qwen/Qwen3-ASR-0.6B`
- 时间戳：`Qwen/Qwen3-ForcedAligner-0.6B`
- 用途：默认档位，优先保证 Windows 本机和 6GB 显存机器跑通。

## 精准模型

- ASR：`Qwen/Qwen3-ASR-1.7B`
- 时间戳：`Qwen/Qwen3-ForcedAligner-0.6B`
- 用途：准确率优先，资源占用更高。

## 输出原则

所有模型输出统一进入 `TranscriptDocument`：

- `source_file`
- `duration_sec`
- `language`
- `model`
- `segments`

每个 segment 只包含 `index`、`start_sec`、`end_sec`、`text`。不生成总结、纪要、摘要、说话人标签或语义分类。
