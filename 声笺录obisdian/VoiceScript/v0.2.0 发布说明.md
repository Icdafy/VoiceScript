# VoiceScript v0.2.0 发布说明

## 版本定位

v0.2.0 是从零重写版本，不沿用 v0.1.4 的任何源码。

## 已实现

- Windows 桌面端上传音频转文字。
- UI 参考上传图片重建，包含左侧导航、中央上传区、右侧设置、最近文件表。
- 黑色 / 白色主题切换。
- Qwen3-ASR 标准模型与精准模型双档位。
- Qwen3-ForcedAligner 时间分段。
- TXT、Markdown、SRT、JSON 导出。
- Obsidian 知识库同步说明。

## 暂不包含

- 会议纪要。
- 总结。
- 语义整理。
- 说话人分离。
- 实时麦克风转写。

## 发布文件

- `VoiceScript-v0.2.0-windows-x64.zip`
