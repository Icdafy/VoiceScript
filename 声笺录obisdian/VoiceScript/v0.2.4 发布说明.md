# VoiceScript v0.2.4 发布说明

## 性能 / 设备

- 新增「运算设备」指示：显示使用 GPU 加速还是 CPU 运算。
- 转录新增「预计剩余时间」估算。
- 支持 GPU 加速：安装 CUDA 版 PyTorch（如 `torch==2.12.1+cu126`）后自动使用 NVIDIA 显卡，否则回退 CPU。
- 为 6GB 显存笔记本 GPU 下调标准模型推理 batch（8 → 4）。

## 界面

- 深色模式改为纯黑背景，卡片近黑。
- Windows 标题栏在深色模式下同步为深色（不再有顶部白条）。

## 说明

- 打包的 .exe 捆绑的是 CPU 版 torch，安装包始终走 CPU；从源码（已装 CUDA torch）运行才用 GPU。

## 暂不包含

- 会议纪要、总结、语义整理、说话人分离、实时麦克风转写。

## 发布文件

- `VoiceScript-v0.2.4-windows-x64.zip`
